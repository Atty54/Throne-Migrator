# Throne-Migrator

Migrating from DB Beta to DB Release in Throne

🇬🇧 English version: [README.md](README.md)

---

## Описание

Данная программа предназначена для миграции с формата DB в программе Throne бета-версий в формат DB программы Throne релизной версии.

---

## Инструкция по использованию

1. Разово запустите скачанную релизную версию Throne.
2. Закройте её, убедившись, что она не осталась висеть в трее.
3. Запустите **Throne-Migrator**.
4. Нажмите кнопку **Browse** напротив:
   - **Path to old Throne DB** → выберите файл `throne.db` бета-версии Throne
   - **Path to new Throne DB** → выберите файл `throne.db` релизной версии Throne
5. Убедитесь, что оба поля заполнены.
6. Нажмите кнопку **Go**.
7. Дождитесь окончания работы программы (появится сообщение `Complete`).
8. Закройте программу.
9. Запустите релизную версию Throne и проверьте, что данные успешно перенесены.

---

## Примечания

- Убедитесь, что оба файла базы данных доступны.
- Рекомендуется сделать резервные копии перед миграцией.
